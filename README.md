# OpenIJ

OpenIJ est un projet open source permettant de communiquer avec certaines imprimantes Canon PIXMA via leur interface réseau.

## Objectifs

- Détecter les imprimantes Canon compatibles
- Lire leur état
- Effectuer les opérations de maintenance
- Remplacer les fonctions essentielles de Canon IJ Toolbox
- Fournir un outil léger, rapide et documenté
